# Snake Game

A classic Snake game implementation in Python using Pygame.

## Features

- Classic snake gameplay
- Score tracking
- Collision detection
- Food spawning
- Smooth controls

## Requirements

- Python 3.x
- Pygame library

## Installation

1. Make sure you have Python installed on your system
2. Install Pygame using pip:
   ```
   pip install pygame
   ```

## How to Play

1. Run the game:
   ```
   python snake.py
   ```
2. Use arrow keys to control the snake:
   - ↑ (Up Arrow): Move up
   - ↓ (Down Arrow): Move down
   - ← (Left Arrow): Move left
   - → (Right Arrow): Move right
3. Eat the red food blocks to grow and increase your score
4. Avoid hitting the walls or the snake's own body

## Game Rules

- The snake grows longer each time it eats food
- Game ends if the snake hits the walls or itself
- Score increases with each food item eaten
